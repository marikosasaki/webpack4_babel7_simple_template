# Usage

## setup

install npm

```sh
yarn
```

## branch

|      branch      |          description          |
| ---------------- | ----------------------------- |
| master           | no preprocessor               |
| feature/extended | pug,scss,postcss,autoprefixer |


## command

webpack --mode development

```sh
yarn dev
```

webpack --mode production

```sh
yarn build
```

start webpack-dev-server

```sh
yarn start
```
